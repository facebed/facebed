Facebed

A better Facebook embed provider for messaging apps.

Usage:

Replace facebook.com with facebed.com or use s/book/bed in Discord.
Append /text to the end of any supported URL to render a text-only version.

Features:

Currently supported URL schemes include: 
/:user/posts/:(id|hash)
/share/p/:hash
/permalink.php?story_fbid
/story.php?story_fbid
/groups/:id/posts/:(id|hash)
Reels are supported. Shared videos and reels are not perfectly supported.
